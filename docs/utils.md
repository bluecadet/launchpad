## Logging

Logging is based off of WinstonJs - [NPM package](https://www.npmjs.com/package/winston), [Github](https://github.com/winstonjs/winston).

For reference, logging levels:

```
{
  error: 0,
  warn: 1,
  info: 2,
  http: 3,
  verbose: 4,
  debug: 5,
  silly: 6
}
```
