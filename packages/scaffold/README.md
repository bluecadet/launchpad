# @bluecadet/launchpad-scaffold

Windows system configuration tools for interactive media installations. This package helps automate the setup of Windows machines for kiosk and exhibit environments.

## Documentation

For complete documentation and guides, visit:
[Launchpad Documentation](https://bluecadet.github.io/launchpad/)

## Features

- Automated Windows kiosk setup
- System optimization and lockdown
- Power settings management
- Startup task configuration
- User interface customization

## Installation

```bash
npm install @bluecadet/launchpad-scaffold @bluecadet/launchpad-cli
```

## Basic Usage

```bash
npx launchpad scaffold
```
